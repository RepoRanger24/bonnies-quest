# Bonnie's Quest

A simple browser-based cozy mystery adventure inspired by classic point-and-click games like King's Quest.

## Version 1 features

- Title screen with intro story text
- Start Quest button
- Bonnie character introduction
- First location: Mabel's Antiques
- One clickable clue item
- Simple inventory area
- Responsive desktop-friendly design

## How to run

Open `index.html` in any modern web browser.

For a local development server, run:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
